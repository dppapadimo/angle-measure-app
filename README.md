# 📐 Angle Measure Pro

Mobile-first εφαρμογή για μέτρηση γωνιών πάνω σε εικόνα.

## Features
- Upload εικόνας
- Zoom + pan
- Πολλαπλές γωνίες
- Drag points
- Live calculation
- Labels A, B, C...
- Export εικόνας με annotations

## Usage
1. Upload εικόνα
2. Tap 3 σημεία (2ο = κορυφή)
3. Σύρε για διόρθωση
4. Save image

## Controls
- Mouse wheel → zoom
- Drag → pan
