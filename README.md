# Angle Measure App 📐

Mobile-friendly εφαρμογή για μέτρηση γωνίας πάνω σε εικόνα.

## Features
- Upload εικόνας
- Pinch zoom & pan
- 3 draggable σημεία
- Υπολογισμός γωνίας (<180°)
- Live update

## Usage
1. Upload εικόνα
2. Tap 3 σημεία (2ο = κορυφή)
3. Σύρε τα σημεία για ακρίβεια
